# Nassim Portfolio

## Project Description
A premium, dark‑themed, glassmorphism‑styled personal portfolio website for **Nassim Elbadri**, showcasing full‑stack development, AI, and cloud projects. The site is built with Vite, vanilla JavaScript, and CSS, focusing on performance, SEO, accessibility, and a recruiter‑friendly experience.

## Features
- Modern dark theme with glassmorphic UI
- Smooth scroll, animated background shapes, and subtle hover effects
- Hero section with availability badge
- Interactive contact card with copy‑to‑clipboard email
- "Why Work With Me" value‑prop grid
- Detailed project showcase cards (category, status, impact, tech stack)
- Timeline of professional journey
- Responsive design (mobile‑first) with breakpoints at 992px and 768px
- Comprehensive SEO metadata (Open Graph, Twitter cards, JSON‑LD)
- Accessible markup with ARIA labels and semantic HTML

## Technologies Used
- **Vite** (build tool)
- **HTML5**, **CSS3** (custom variables, animations, glassmorphism)
- **JavaScript (ES6+)**
- **Git** for version control
- **Node.js** (for development server & tooling)

## Installation
```bash
# Clone the repository
git clone https://github.com/nassimelbadri/Nassim-Portfolio.git
cd Nassim-Portfolio

# Install dependencies
npm install
```

## Development
```bash
npm run dev   # Starts Vite dev server (http://localhost:5176)
```
Open the URL in a browser and the site will live‑reload on changes.

## Production Build & Deployment
```bash
npm run build   # Generates optimized static files in ./dist
```
Deploy the contents of the `dist` folder to any static hosting platform (Vercel, Netlify, GitHub Pages, etc.).

### Deploy to Vercel (example)
1. Sign in to Vercel and import the GitHub repository.
2. Set the **Build Command** to `npm run build` and the **Output Directory** to `dist`.
3. Click **Deploy** – Vercel will automatically build and host the site.
